# STATNLP Mention Hypergraph Service

## Models
- mentionHypergraph-small.model is not accurate
- mentionHypergraph.model is quite slow
- [Download models](https://drive.google.com/open?id=1dMY1ebw3o7Kv7TkEx6Z991zbA5apKfrN)
